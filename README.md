# Stathat

StatHat client library for Elixir.

## Installation

The package can be installed as:

  1. Add stathat to your list of dependencies in `mix.exs`:

        def deps do
          [{:stathat, "~> 0.0.1"}]
        end

  2. Ensure stathat is started before your application:

        def application do
          [applications: [:stathat]]
        end
